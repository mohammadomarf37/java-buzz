# Java Buzz Coffee Shop Website

A modern, responsive website designed for "Java Buzz", a new coffee shop.

## Features
- **Responsive Design:** Works on Mobile, Tablet, and Desktop.
- **Interactive Map:** Uses Leaflet API to show location.
- **Dynamic Blog:** Loads 10 blog posts from a JSON file.
- **Menu Section:** Clean layout with pricing.
- **SEO Optimized:** Includes Meta tags, Robots.txt, and Sitemap.xml.

## Technologies Used
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (Fetch API)
- Leaflet.js (Maps)

## How to Run
1. Clone this repository.
2. Open `index.html` in your web browser.
